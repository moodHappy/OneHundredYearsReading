# Macondo Matrix (枢纽)

**Macondo Matrix** 是一个基于 GitHub Pages 实现的轻量级、去中心化「全栈语法解构库」。它利用 GitHub API 作为数据库，将碎片化的英语学习笔记（HTML 格式）存储于云端，并提供类似原生 App 的日历检索、深度搜索与实时编辑体验。

## 核心架构
本项目无需后端服务器，通过 Python 预编译脚本 `generate_index.py`，将分布在 `docs/` 目录下的所有 HTML 笔记自动索引化，生成单页面枢纽（`index.html`）。

*   **前端**：原生 JS + CSS 实现的响应式滑动界面（Viewport）。
*   **后端**：GitHub REST API（实现读、写、删、查）。
*   **交互**：支持 Markdown 渲染、云端静默同步、即时搜索。

## 功能特性
- **云端矩阵**：所有解构笔记物理存储在仓库中，支持跨设备同步。
- **极致检索**：通过输入关键词，瞬间完成对全库（标题+原文内容）的全文检索。
- **日历枢纽**：直观的日历视图，通过红点标记显示历史学习轨迹。
- **沉浸式解构**：针对每条记录提供 Markdown 渲染支持，支持双击/手势快速进入编辑模式。
- **无感同步**：修改后的内容会自动触发 GitHub API `PUT` 请求，无需手动保存。
